# 📁 File Management System in Python

## 📌 Overview

The File Management System is a command-line Python application that enables users to perform basic file operations such as creating, reading, updating, renaming, appending, and deleting files. The application provides a simple menu-driven interface and displays all available files and folders before performing operations.

## 🚀 Features

- Create a new file
- Read file contents
- Rename existing files
- Overwrite file contents
- Append data to files
- Delete files
- Display all files and folders in the current directory
- Exception handling for error management

## 🛠️ Technologies Used

- Python 3.x
- pathlib Module
- os Module

## 📂 Project Structure

## 📖 How It Works

### 1. Create File
- Enter a file name.
- Add content to the file.
- File is created successfully.

### 2. Read File
- Select an existing file.
- View file contents.

### 3. Update File
Options:
- Rename file
- Overwrite file content
- Append new content

### 4. Delete File
- Select a file.
- Delete it permanently.
